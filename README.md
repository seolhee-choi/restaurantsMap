## 회사 근처 맛집 지도(성수동)
자사 직원에 한해 등록 및 조회 가능하며 session으로 User 필터링

<br></br>
## 🛠️ 실습환경

- JAVA, JavaScript(ES6 포함), Jquery
- IntelliJ
- 카카오맵 API
  
<br></br>
## 🔑 KEY Point
  - 카카오맵 API를 통해 회사 근처에 식당을 맛집으로 등록 가능
  - 등록된 맛집에 대한 리뷰(해시태그, 별점, 사진첨부)를 남길 수 있는 서비스
